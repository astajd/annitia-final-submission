## Notes

- Same feature sets as Phase 3.10 / 3.12. Same hyperparameters. Only the training scheme changes: each component is now trained on the full 1253-row training set with no fold splits.
- Test predictions are produced by a single fit per component instead of the 15 fold-fits that the anchor averages.
- We do not invent an OOF estimate for the full-retrain model — there is no honest way to compute one. Recommendation rests on test-prediction rank-correlation with the anchor.
